# News-Click-through-behavior
## Overview

This repository contains the code developed for my MSc Business Analytics dissertation at Trinity College Dublin.

The project investigates the factors influencing user click-through behaviour in digital news recommendation systems using the Microsoft News Dataset (MINDsmall). The analysis combines behavioural analytics, natural language processing, and temporal analysis to examine how different factors influence user engagement with online news.

The repository includes the analytical code used to answer four research questions, together with the exploratory data analysis (EDA) performed during data preparation.

## Repository Contents

This repository contains the following notebooks:

EDA – Exploratory data analysis and descriptive statistics of the processed dataset.

RQ1 – User Segmentation – Behavioural segmentation of readers using K-Means clustering and cluster profiling.

RQ2 – Headline Keyword Analysis – Analysis of news categories, headline keywords, TF-IDF feature extraction, and logistic regression.

RQ3 – Emotional Framing Analysis – Sentiment analysis of headlines using VADER and TextBlob together with clickbait feature analysis.

RQ4 – Temporal Pattern Analysis – Investigation of hourly engagement patterns, time-of-day effects, and category-specific publishing windows.

## Dataset

The analyses are based on the MINDsmall dataset released by Microsoft.

## Main Methods

The project applies a range of data analytics and machine learning techniques, including:

Data preprocessing and integration

Exploratory Data Analysis (EDA)

Feature engineering

Click-Through Rate (CTR) analysis

K-Means clustering

TF-IDF vectorisation

Logistic Regression

VADER sentiment analysis

TextBlob sentiment analysis

Temporal feature extraction

Data visualisation

## Repository Purpose

This repository is intended to support the reproducibility of the dissertation by providing the analytical code used throughout the study. The notebooks correspond directly to the empirical analyses presented in the dissertation.

The repository does not include the dissertation document, figures, or the MIND dataset.
